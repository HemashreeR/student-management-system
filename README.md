# Student Management System

A full-stack CRUD application built using React.js, Node.js, Express.js, and MongoDB to manage student records.

## Features

* Add Student
* View Students
* Update Student Details
* Delete Student
* Search by Name or Email

## Tech Stack

* React.js
* Context API
* Axios
* Node.js
* Express.js
* MongoDB

## Installation

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm start
```
